# Shopify App Store keyword analyzer Chrome extension

## Description

API build with NestJS for [chrome extensions](https://github.com/themarchenko/shopify-app-store-kw-ext).

## Project setup

```bash
$ yarn install
```

## Compile and run the project

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Run tests

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
